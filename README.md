# HandTrackingAirMouse
A semi structured repository for a small project where i attempt to create a tool which allows for navigation using only hand gestures and movement

use python 3.10, .ater versions are not compatable with media pipe

try to have a darker clear backhround for efficient handtracking

# Controls

**left hand**

victory - right click

fist - left click

thumbs up - scroll up

thumbs down - scroll down

**right hand**

index finger - cursor movement
